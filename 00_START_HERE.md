## 🎯 Agent Architecture

The agent follows this reasoning loop:

```
START (Understand Task)
  ↓
THINK (Plan approach)
  ↓
[Multiple THINK steps for deeper reasoning]
  ↓
TOOL (Execute action: create files, folders, etc.)
  ↓
OBSERVE (Process results)
  ↓
[Loop back if task not complete]
  ↓
OUTPUT (Task complete - present results)
```
