### 2.1 Prepare

1. Get OpenAI API key: https://platform.openai.com/api-keys
2. Add to `.env` file:
   ```
   OPENAI_API_KEY=sk-your-actual-key-here
   ```
3. Test: `npm run demo` works properly

### 2.2 Recording (Use one option)

**Option A: Windows Game Bar (Easiest)**

1. Open terminal with project
2. Press `Win + G`
3. Click "Record" button
4. Run: `npm run demo`
5. Let it complete
6. Stop recording
7. Video auto-saves to Videos folder

**Option B: OBS Studio (Free)**

1. Download: https://obsproject.com/
2. Select screen/window to record
3. Hit record
4. Run: `npm run demo`
5. Stop and save video
